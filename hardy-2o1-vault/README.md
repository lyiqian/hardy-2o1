## 开叫
### 一阶花色开叫
- 5 Major, best minor
- 22法则：大牌点 + 最长两套张数 + 防守赢墩 >= 22
	- 防守赢墩：A=1; K=0.5; AK=2; AQ=1.5;  KQ=1
- 第四家开叫要足点
### 无将开叫
**1NT**：15-17P，均型
- 可有5张高花或6张低花

**2NT**：20-21P，均型
- 可有5张高花或6张低花
### 人工强开叫
**2♣**，*无奈*最强开叫｜p182
- 平均牌型22p↑，或 不平均牌型9赢墩↑。
- 注：有9赢墩且可以一阶开叫的牌，优先一阶开叫
### 弱二开叫
**2♢/2♡/2♠，弱二**｜p178
- 6-11p，6张，无其他4张高花
    - 第一、二家：6-11hcp，保证[AKQ, 2] or [AKQJT, 3]
    - 第四家：10-13hcp，好牌好套
### 阻击叫
**3♣ ~ 5♢，阻击叫**｜p175
- 234法则；无其他高花；防守赢墩<=1
- 应叫
	- *加叫*: 加深阻击
	- *新花色*: 5张↑，逼叫（根据234法则判断）

**5♡及以上**，自然
- 承诺所需赢墩，且将牌外无输墩
- 应叫
	- *加一(两)阶*：开叫花色上有一(两)大，其它花色上的大牌几乎无用。

**3NT，赌博性**｜p440
- 七张坚固低花套，其他花色无止
- 应叫
    - *Pass:* 其他花色有止
    - *4♣:* 逃，pass or correct
    - *4♢:* 半逃，pass or 5♣
    - *5♣:* 双高有4个快速赢墩，pass or correct

## 附录

**[1, AKQ]** 表示AKQ中至少有1张。

**优先八张高花局**是基本原则。
###### 名句引用
- 配合。带来（更多）赢墩的是配合，而不是点
- 加叫。（尤其在争叫时）永远不要错过加叫同伴的机会｜p201
- 平衡跳叫。平衡时没有（**阻击**）弱跳叫；所有的跳叫都表示好牌 ｜ p206
	- 因为阻击是防止敌人进局，平衡位置阻击对手最好的方法就是pass
- 4－4配优于5－3配（将牌质量也要考虑）

## 一般性约定

### 通识
- 逆叫：第一套更长，逼叫
- **应叫人**新花：通常是逼叫，原花通常不逼叫（即便跳原）
- 进局：局况有局的情况下，成功概率超过37.5%的局是应该叫的
- 低花进局：需要额外实力
- 迂回＝特别

### 扣叫
- 扣叫RHO争叫花色：首先表强支持，次之可能寻求止张
- 扣叫敌两门花色之一：自然有止
- 低花配合后，3NT之前的扣叫：表示有止，暗含对未叫花色的担忧
- 满贯兴趣时，不扣同伴花色的单缺 ^161d99

### 示攻加倍与对抗
- 我方虚叫被示攻Dbl后：Rdbl示弱，Pass示强 ^7531c5
	- Stm 2♣ dbl: rdbl 无高无止; pass 无高有止; 2d 实叫无止
	- 转移 dbl: rdbl 无配无止; pass 无配有止; 接受转移, 有配合
- 对二阶虚叫示攻dbl的一般性要求：5张4点3大

### 其他
- 总墩数定律： 局部定约争夺，基于联手将牌张数-6
- 联手总HCP>23时，“定约属于我方”
	- 比如，限制性加叫后，2N邀请后，等
	- 不能轻易放对方打成局定约，有*逼叫性pass*
- 有两个未叫或敌方花色时，自然描述或表示止张
- 没找到配合时，空间更重要 ^02ced4
	- 比如 1M1n 2m：长度一样时，忽略强度叫2c
### 满贯
- Splinter：四支持，单缺，加上同伴的低限有25p
	- 小Splint，如跳逆叫和低一级跳叫：可减少3p
	- 原理之一：25p可打成四阶，若能够通过单张将吃两墩，可能有六； [https://g.co/kgs/vFBwAta](https://g.co/kgs/vFBwAta)
- 示量 4NT：至少一方叫NT显示过均型
